## 📦 快速开始

### 方式一：Docker 快速体验（参考）

```bash
# 拉取镜像
docker pull yesapi/yesapi-java-pro:latest

# 启动（需要提前准备 MySQL 和 Redis）
docker run -d \
  -p 8080:8080 \
  -e DB_HOST=your-mysql-host \
  -e DB_NAME=yesapi \
  -e DB_USER=root \
  -e DB_PASS=yourpassword \
  -e REDIS_HOST=your-redis-host \
  --name yesapi-pro \
  yesapi/yesapi-java-pro:latest
```

> 📌 启动后访问 `http://localhost:8080` 进入管理后台

### 方式二：Shell 一键部署命令（参考）

```bash
docker-compose up -d
```

## ❓ 常见问题

<details>
<summary>Q: YesApi Pro 和 Postman / Swagger 有什么区别？</summary>

Postman 和 Swagger 是**调试和文档工具**，本质上是帮你写文档、测接口的。
YesApi Pro 是**开发和运营平台**，帮你低代码生成 API、管理权限、收费变现、私有部署。
两者不是竞争关系——很多企业用 YesApi Pro 开发 API，用 Postman 做接口联调测试。
</details>

<details>
<summary>Q: 源码质量怎么样？可以放心二次开发吗？</summary>

7 年持续迭代，500+ 企业生产环境验证，日均千万级接口调用。
源码提供详细注释和开发文档，架构清晰（Spring Boot / Laravel 标准架构），
绝大多数二次开发需求可以在不改动核心逻辑的情况下通过插件或扩展点实现。
</details>

<details>
<summary>Q: 私有部署后，厂商能访问我们的数据吗？</summary>

完全不能。私有部署意味着 YesApi Pro 运行在你的服务器上，
我们不保留任何后门，也没有回传逻辑。购买源码版后，你可以完整审计每一行代码。
</details>

<details>
<summary>Q: 支持国产信创环境吗？</summary>

Java 版完全支持：
- CPU：鲲鹏、飞腾
- 操作系统：麒麟 V10、统信 UOS
- 数据库：达梦 DM8、OpenGauss、人大金仓
- 中间件：东方通 TongWeb、金蝶 Apusic

如有特殊信创需求，可提前告知，我们提供适配评估。
</details>
