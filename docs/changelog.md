以下是YesApi Pro版本的统一术语定义：  

| **术语**          | **定义**                                                                 | **示例**                              |  
|---|---|---|  
| 接口服务名称      | 接口中文标题名称，简要描述接口功能或用途。                                      | 订单查询接口                     |  
| 接口功能描述      | 用于描述接口功能或用途。                                      | 关于接口功能的一段文字内容描述。                     |  
| 接口路由          | 接口的访问相对路径，用于定位接口功能，唯一标识不同的接口服务。                                        | `/api/user/profile`                  |  
| 接口套餐名称          | 接口服务的购买套餐，有时简称：套餐名称，涵盖余额充值等订单套餐。                                            | 对应管理后台配置的接口套餐名称，例如：接口流量包套餐名称、组合优惠套餐名称、接口免费试用套餐名称等。                     |  
| 接口请求地址      | 接口的完整访问请求地址，包含域名和路由和参数，主要是提供给客户端程序调用。                                | `https://api.yesapi.cn/api/user/profile` |  
| 接口请求方式      | 接口的HTTP请求方式。                                | 例如：GET、POST |
